Mengenal Semantic Elements Di HTML5

  • Article

    Ketika kita ingin membuat layout untuk menampilkan block artikel-artikel yang ada di website kit, tag article bisa kita gunakan dengan element header dan footer.

    Karena tag article memiliki type block, artinya element ini memilik lebar sesuai lebar browser jika kita tidak mentukan lebarnya

  • Aside

    Element ini bisa kita gunakan untuk melengkapi layout utama. Misal kita memiliki halaman blog maka kita bisa menambahkan aside di bagian bawah postingan untuk menjelaskan penulis artikel.

    Karena asie memiliki type/sifat block maka lebarnya sesuai page jika kita tiak menentukan lebarnya.
